# Set up
Fill in the configuration .env.template file and rename it to .env
Create a virtual environment and run 
    pip install -r requirements.txt

# Run examples
Run:
    cd [example folder: 1_prompt or 2_states or 3_agents]
    python main.py

Enjoy!
